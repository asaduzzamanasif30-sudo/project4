# 🛒 GadgetBD — Responsive Tech E-commerce Landing Page

A high-fidelity **Figma to HTML** conversion project. GadgetBD is a modern, clean, and fully responsive e-commerce landing page designed for selling the latest tech gadgets and accessories.

## ✨ Key Features
- ✅ **Figma to HTML:** Precise implementation of the original Figma design with attention to spacing, typography, and color accuracy.
- ✅ **Fully Responsive:** Optimized for all screen sizes, including 4K monitors, laptops, tablets, and mobile devices.
- ✅ **Modern UI Components:** Includes a sticky navigation bar, categorized shop sections, and detailed product cards.
- ✅ **Interactive Elements:** Hover effects on buttons and product cards for an enhanced user experience.
- ✅ **Optimized Assets:** High-quality imagery and SVG icons for fast loading and sharp visuals.

## 📂 Project Sections
1. **Header:** Clean navigation with search, wishlist, and cart icons.
2. **Hero Banner:** Attractive "Latest Tech Gadgets" slider/banner with call-to-action buttons.
3. **Category Section:** Grid layout for quick navigation to Smartphones, Laptops, Audio, etc.
4. **Flash Sale:** Time-sensitive product deals with discount badges.
5. **Featured Products:** A showcase of top-rated technology items with price and ratings.
6. **Footer:** Comprehensive site map, customer service links, and payment method icons.
